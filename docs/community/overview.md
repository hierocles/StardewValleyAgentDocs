Stardew Valley has a very friendly modding community. If you have
questions about using or creating mods, come talk!

## Where to Talk

### Reddit

[r/SMAPI](https://www.reddit.com/r/SMAPI) is the official SMAPI
subreddit for asking questions, getting help, and discussing SMAPI +
mods + content packs.

### Discord

The [Stardew Valley Discord](https://discord.gg/stardewvalley) (no
microphone, audio, or download needed) has...

- `#modded-stardew` and `#making-mods-general`: discuss or ask questions
  about using or creating mods. Modders also discuss their work, chat,
  and collaborate here.
- `#modded-tech-support`: get help with modding issues.
- `#vanilla-tech-support`: get help for issues with the game itself
  (like crashes, corrupted saves, or the game not launching).
- General channels to discuss the game and chat with other players.

### Official Forums

The [Stardew Valley Forums](https://forums.stardewvalley.net/) have the
following sections:

- Modding Discussion & Creation: discuss or ask questions about using or
  creating mods.
- Mod Support: get help with modding issues.

### Mod Sites

If you need help with a specific mod, consider asking on the mod page
you downloaded it from. (For example, see if you downloaded it from
there.)
