## Modders

Here are the users on Discord who have the [mod author
role](https://gist.github.com/Pathoschild/927aa919a95cf0ddae542749bc9d9847#mod-author-roles).
If you created a mod and reached level 25 on Discord, feel free to
<a href="#Discord" class="wikilink"
title="ask in #making-mods-general">ask in #making-mods-general</a>, or
DM *@Bouncer#9811*! We'll check if you qualify, help you add yourself to
the list, and give you the `Mod Author` role. (See
<a href="Modding_Community_entry" class="wikilink"
title="instructions for adding an entry">instructions for adding an
entry</a>, but please don't add yourself to the wiki before asking.)
