## For players

See the [mod compatibility list](https://smapi.io/mods) to check your
mods!

To update your mods:

1.  For each mod you have installed, check the [mod compatibility
    list](https://smapi.io/mods) for instructions. If it's not listed,
    try updating to its latest version.
2.  If a mod is still failing, see
    <a href="Modding_Help" class="wikilink"
    title="Modding:Help">Modding:Help</a> for general help or the
    [unofficial update
    thread](https://forums.stardewvalley.net/threads/unofficial-mod-updates.2096/)
    if it's marked incompatible.

## For mod authors

The data for the [mod compatibility list](https://smapi.io/mods) is
maintained in the on GitHub. Contributions are welcome if you notice
something is outdated or missing!
