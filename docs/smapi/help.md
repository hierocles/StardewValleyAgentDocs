Stardew Valley has a very friendly community. If you need help, this
page is for you!

## Get help with…

### Using mods

Before you ask:

:# Check the
<a href="Modding_Player_Guide_Getting_Started" class="wikilink"
title="player&#39;s guide to mods">player's guide to mods</a> and
<a href="Modding_Player_Guide_Troubleshooting" class="wikilink"
title="troubleshooting page">troubleshooting page</a>, many questions
are answered there.

:# Need help with a specific mod? Try asking on its mod page or
[checking the mod compatibility list](https://smapi.io/mods/) first.

:# See
<a href="#Get_help" class="wikilink" title="what to include">what to
include</a> when asking for help.

Where to ask:

:# The [Stardew Valley Discord](https://discord.gg/stardewvalley) (no
mic, audio, or download needed) has a `#modded-tech-support` forum for
help installing/using mods, and a `#modded-farmers` channel for
discussing mods in general.

:# [r/SMAPI on Reddit](https://www.reddit.com/r/SMAPI) for questions and
discussions with the community for SMAPI, Content Patcher, and other
mods and content packs.

:# The [Mod Support
section](https://forums.stardewvalley.net/forums/mod-support.29/) on the
official forums.

## How to…

### Get help

See the above sections for where to get help. Make sure you provide info
to help find the problem, including…

- your platform (*e.g.,* Linux, Windows, Android, etc);
- your game version;
- your [SMAPI log](https://smapi.io/log/) (if playing with mods);
- repro steps (instructions to cause the issue);
- screenshots of the issue (if relevant).

### Share your log

*(Only applies if you use mods.)*

The SMAPI log has a lot of useful info; include it even if you don't see
an error!

To share your log:

1.  [Upload it to smapi.io/log](https://smapi.io/log/) (see that page
    for instructions).
2.  Copy the URL it gives you after uploading.
3.  Include the URL when asking for help.

### Share screenshots

1.  Take a screenshot of the issue. See instructions for [PC (Linux,
    macOS, Windows)](https://www.take-a-screenshot.org/), [mobile phone
    (Android or iOS)](https://www.take-a-screenshot.org/), and [Steam
    Deck (portable
    console)](https://www.pcgamesn.com/steam-deck/screenshot).
2.  Upload it to a site like [Pasteboard](https://pasteboard.co/).
3.  Include the URL when asking for help.

*Note: when asking for help on Discord, you can also paste or drag the
file into Discord directly. Please only do this with screenshots as a
link to your log is more helpful than uploading the file to Discord.*

### Share files

If you need to share files:

1.  Upload it to a site like [catbox.moe](https://catbox.moe/).
2.  Include the URL when asking for help.
